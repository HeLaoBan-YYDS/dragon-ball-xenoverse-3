タイムパトロールに戻る準備はできていますか？待望の次回作の登場を控え、ファンはすでに**ドラゴンボール ゼノバース3のキャラクター（dragon ball xenoverse 3 characters）**の膨大なラインナップに胸を躍らせています。カスタマイズ可能なヒーローから伝説の戦士まで、私たちが**ドラゴンボール ゼノバース3のキャラクター**と関わる方法は、ゲームプレイの全面的なオーバーホールによって劇的に変化します。前作の出来事から148年後を舞台とする本作は、革新的なメカニクスと、伝説的な鳥山明氏によって築かれた深いストーリーの基盤により、アリーナ対戦アクションのジャンルを再定義しようとしています。

ディンプスが開発し、バンダイナムコエンターテインメントがパブリッシングを行う本作は、過去作の古いビジュアルスタイルから脱却し、近年のアニメ作品と調和する、非常に流動的でモダンな美学を採用しています。プレイヤーは、活気に満ち、飛行ビークルが行き交い、バトルで共闘してくれるお馴染みのヒーローたちが暮らす、巨大な垂直都市「ニューシティ」を探索することになります。対戦プレイが好きなプレイヤーも、カジュアルなファンも、この新しいサンドボックスでキャラクターがどのように機能するかを理解することが、ゲームをマスターするための鍵となります。

## AGE 1000の幕開け：タイムパトローラーの新時代

本作のストーリーはAGE 1000を舞台としており、前作の結末から約1世紀半後の世界となります。ゲームの舞台をAGE 1000に設定することで、ディンプスは複数の時代、タイムライン、そして完全オリジナルのストーリーアークにまたがる、非常に多様な**ドラゴンボール ゼノバース3のキャラクター**を登場させることが可能になりました。鳥山明氏は本作の開発に深く関わっており、AGE 1000のストーリー基盤を構築し、男女のカスタムプレイヤー主人公、ブルマ、そしてブレットという名の新しいヒーローを含む、いくつかの主要キャラクターのデザインを手掛けました。

無機質なタイムパトロール広場で目覚める代わりに、プレイヤーはニューシティにあるリアルで生活感のあるアパートから旅を始めます。一歩外に出ると、伝説の戦士たちが隣人として暮らす広大な大都市が広がっています。例えば、初期のゲームプレイ映像では、アパートの近くにサイヤ人襲来編のベジータがたむろしていたり、未来のトランクスがドアのすぐ外で警備に立っていたりする様子が確認できます。街は非常にインタラクティブで、ショップやミッションデスクがあり、空には操作可能なビークルが飛び交うなど、まさに「MMOライト」な体験を提供しています。

## ソウルボンディング＆ソウルエクスチェンジ：戦闘の革命

本作で導入される最も画期的な機能の一つが、『ドラゴンボール ファイターズ』や『ドラゴンボール ザ ブレイカーズ』のメカニクスから着想を得た「ソウルボンディング（Soul Bonding）」のコンセプトです。このシステムにより、カスタムキャラクター（CaC）は伝説のZ戦士たちと深い精神的な絆を結ぶことができます。戦闘中、この絆は「ソウルエクスチェンジ（Soul Exchange）」と「ソウルアシスト（Soul Assist）」という2つの異なる形で発現します。

ソウルエクスチェンジを通じて、プレイヤーは文字通りお馴染みのヒーローの魂をカスタムキャラクターに直接宿らせることができます。発動すると、ヒーローの魂がアバターと融合する見事なカットシーンが流れます。制限時間内（初期のゲームプレイビルドでは41秒に設定）は、そのキャラクターを完全に操作できるようになり、そのキャラクターの格闘スタイル、必殺技、究極技をすべて使用できるようになります。ソウルエクスチェンジが発動している間は、キャラクターの最大体力が劇的に増加し、頭上にはカプセルコーポレーション風のユニークな天使の輪（ハロー）が現れます。

例：地球人のカスタムキャラクター「セン」は、未来のトランクスとソウルエクスチェンジを行うと、体力が即座に60,900 HPから79,200 HPへと上昇します。

このメカニクスは、それぞれ異なる**ドラゴンボール ゼノバース3のキャラクター**をメインソウルとして運用しているチームメイトと連携する際に、深い戦術的レイヤーをもたらします。ソウルエクスチェンジのタイマーが切れると、ソウルアイコンは戦闘中にゆっくりと再充填され、試合が長引けば複数回変身を発動させることが可能です。

| 機能 | ソウルエクスチェンジ | ソウルアシスト |
| :--- | :--- | :--- |
| **主な機能** | キャラクターの完全交代と技セットの切り替え | 1回限りのアイコニックな攻撃の強化 |
| **効果時間** | 制限時間あり（例：41秒、アップグレード可能） | 即時発動 |
| **ステータスバフ** | 大幅な体力増加、物理ダメージ上昇 | クールダウン短縮、局所的なチームバフ |
| **視覚的インジケーター** | 頭上のカプセルコーポレーション風の輪 | アシストするヒーローの一時的な残像表示 |
| **チャージ方法** | 戦闘中のパッシブゲージの低速回復 | クールダウンベースのチャージ（例：最大2チャージ） |

一方、ソウルアシストを使用すると、絆を結んだヒーローを呼び出して、素早く強力な一撃を放つことができます。例えば、サイヤ人襲来編のベジータを召喚して、自身のギャリック砲を一時的に強化し、ダブルチャージすることができます。これらのアシスト技はチャージ制のクールダウンシステムで動作するため、クールダウン短縮を中心にキャラクターを構築したプレイヤーは、これらの強力な攻撃を繰り返し繰り出すことができます。

## カスタムキャラクター（CaC）と格闘スタイルの共有

シリーズの歴史の中で最も要望の多かった機能の一つが、ついに実装されました。本作では、カスタムキャラクターはもはや種族ごとの一般的な基本攻撃モーションに縛られません。その代わりに、お馴染みの**ドラゴンボール ゼノバース3のキャラクター**の技セットや戦闘スタイルを借りることができる「格闘スタイル共有（Fighting Style Sharing）」が導入されます。

未来のトランクスの優雅な剣術アクションが好みなら、その近接攻撃の流れを地球人、サイヤ人、またはナメック星人のカスタムキャラクターにそのまま割り当てることができます。ベジータの攻撃的なインファイトや、アルティメット悟飯の緻密に計算された戦闘スタイルが好みなら、それらをマスターしてアバターに直接装備することが可能です。

カスタムアバターの技セットが制限されていた過去作とは異なり、**ドラゴンボール ゼノバース3のキャラクター**のリストは直接の修行の場として機能します。この機能は、専門化されたスポーツゲームのビルドに似ており、プレイヤーにお気に入りのファイターの物理アニメーションと、カスタムキャラクターのユニークな外観を自由に組み合わせてマッチさせる完全な自由を与えてくれます。

| ステップ | アクション | 説明 |
| :--- | :--- | :--- |
| **1** | **師匠と絆を結ぶ** | ニューシティで伝説のキャラクターたちと交流し、彼らの戦闘スタイルをアンロックします。 |
| **2** | **流れをマスターする** | 戦闘試練や修業ミッションをクリアして、彼らの近接攻撃アニメーションを完全にアンロックします。 |
| **3** | **スタイルを装備する** | カスタマイズメニューに行き、アンロックした格闘スタイルをCaCに適用します。 |
| **4** | **必殺技と組み合わせる** | 新しく習得した近接アニメーションを、互換性のある必殺技と組み合わせます。 |

## 強化された変身とビジュアルのアップグレード

変身システムは大幅な見直しが行われ、コミュニティから長年寄せられていた批判に応えています。過去作では、超サイヤ人に変身するとキャラクターの髪の色は変わるものの、髪の実際の3Dモデルは通常時のままでした。

本作では、変身時のダイナミックな髪のレンダリングが導入されています。カスタムキャラクターが超サイヤ人に変身すると、髪が物理的に逆立ち、完全にユニークで攻撃的なスタイルへと変化します。超サイヤ人3のような高度な形態では、非常にシネマティックなリアルタイムの移行アニメーションが特徴です。カメラがズームインし、キャラクターの後頭部に回り込み、髪が段階的に物理的に伸びていき、お馴染みのフル長さに達する様子をリアルに描きます。

| 変身形態 | 髪の視覚的変化 | 戦闘＆ゲームプレイへの効果 |
| :--- | :--- | :--- |
| **超サイヤ人1** | 通常の髪がダイナミックに逆立ち、クラシックな超サイヤ人のスタイルに変化 | 物理および気弾ダメージが中程度上昇 |
| **超サイヤ人3** | ドラマチックなカメラズームとともに、髪が段階的に長くなる | 物理ダメージが大幅に上昇、気力の回復速度が向上 |
| **ソウルエクスチェンジ（トランクス）** | アバターがカプセルコーポレーションの輪を浮かべたトランクスの姿になる | 技セットの完全な切り替え、最大体力の膨大な増加 |

## 協力バトルとボス戦のメカニクス

マルチプレイヤー体験は、チームの連携と戦略的計画を重視するために大幅にアップグレードされました。戦闘システムにはMMOライトなメカニクスが組み込まれており、プレイヤー同士が攻撃を連携させてブロリーのような強力なレイドボスを封じ込めることができます。

例えば、プレイヤーは攻撃を連携させて、敵の気力ゲージを狙い撃ちして枯渇させることができます。このゲージを枯渇させることに成功すると、ボスは高ダメージの究極技を発動できなくなり、攻撃能力を効果的に無力化できます。さらに、戦闘中にチームメイトが倒された場合、生き残っているチーム全員で救助に駆けつけることができます。複数のプレイヤーが蘇生に参加することで、回復モーションが大幅にスピードアップし、味方をほぼ瞬時に戦線に復帰させることができます。

プレイヤーの体験談や初期のコミュニティレポートによると、ゲームには非常に面白くも実用的な戦闘挙動もあります。敵のボスがチームメイトの1人を掴んで手荒に扱ったり、投げ飛ばしたりするモーションにロックされている間、他のプレイヤーは離れた場所からボスに大ダメージを与え続けることができます。つまり、ボスに狙われることが戦術的なデコイ（おとり）として機能し、ボスのガードが下がっている間にスクワッドが物理打撃や気弾で敵を袋叩きにできるのです。

世界観やコミュニティの議論についてさらに深く知りたい場合は、[Xenoverse 3に関するドラゴンボールWikiのページ](https://dragonball.fandom.com/wiki/Dragon_Ball_Xenoverse_3)をご覧ください。

## よくある質問

### 現在、参戦が確定しているドラゴンボール ゼノバース3のキャラクターは誰ですか？
現在確定している**ドラゴンボール ゼノバース3のキャラクター**の初期ロースターには、クラシックな戦士と完全オリジナルのヒーローが混在しています。プレイアブルキャラクターには、カスタムの男女タイムパトローラーと未来トランクスが含まれます。主なNPCには、ブルマ（AGE 1000）、ブレット（サイヤブレット）、ガンマ1号、ロム、リリカ、タップ、神龍、そしてサイヤ人襲来編ベジータがいます。確定している敵およびボスには、ブロリー、アプール、バトルジャケット、ナバーナ、ナーベル、ラーズベリ、ロベリが挙げられます。

### バトルにおけるソウルエクスチェンジの仕組みはどうなっていますか？
ソウルエクスチェンジは、カスタムキャラクターが未来トランクスのようなお馴染みのヒーローの魂と融合できるシステムです。発動すると、制限時間内（初期設定では41秒）そのヒーローの技セット、必殺技、格闘スタイルを完全に操作できるようになります。この変身により、一時的に最大体力が大幅に増加し、キャラクターの頭上にユニークなカプセルコーポレーションの天使の輪が表示されます。

### カスタムキャラクターはお馴染みのキャラクターの格闘スタイルを使用できますか？
はい！本作には「格闘スタイル共有（Fighting Style Sharing）」システムが搭載されています。ニューシティにいる様々な伝説のキャラクターたちと絆を結び、彼らのもとで修行することで、カスタムキャラクターは彼らの基本的な近接攻撃のコンボやアニメーションを学習し、装備することができます。これにより、キャラクターの選択した種族に関係なく、トランクスの剣術スタイル、ベジータの攻撃的なラッシュダウン、あるいは悟飯の戦術的な戦闘の流れを使って戦うことができます。

### ゲームの発売時期と対応プラットフォームはいつですか？
本作は現在、2027年に日本での発売が予定されています。ディンプスが開発し、バンダイナムコエンターテインメントが販売します。対応プラットフォームは、PlayStation 5、Xbox Series X/S、およびSteam（Microsoft Windows）が確定しています。
