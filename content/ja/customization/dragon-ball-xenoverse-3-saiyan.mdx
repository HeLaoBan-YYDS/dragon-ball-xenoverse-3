## Age 1000におけるサイヤ人：伝承と背景

本作の物語は大きな飛躍を遂げます。舞台となるのは、前作（ゼノバース2）の結末からちょうど148年後にあたる「Age 1000」です。この未来の時代は、ドラゴンボールのストーリーラインの決定的な未来として、鳥山明先生によって緻密に構想されていました。

このタイムラインにおいて、地球は高度に発達した社会へと進化しています。「ニュー・シティ」は、地球人、サイヤ人、ナメック星人、魔人、そしてフリーザ一族が共存する中心的なハブとして機能しています。サイヤ人はもはや、生き残りをかけて戦う絶滅寸前の散り散りになった種族ではありません。代わりに、地球の防衛軍、特に「グレートサイヤマンスクワッド（GSスクワッド）」に統合されています。

### グレートサイヤマンスクワッド（GSスクワッド）
ブルマや、ブレットといった新キャラクターたちの指導のもと、GSスクワッドは平和維持部隊として活動しています。**ドラゴンボール ゼノバース3 サイヤ人**をデザインする際、タイムラインの移行がキャラクターにどのような影響を与えるかを理解することは極めて重要です。あなたは歴史の裂け目から引きずり出された単なる生存者ではなく、世界平和のために自らの潜在能力を開花させるべく、専門の学校に入学した訓練生なのです。

| ゲームタイトル | 主要ハブ | タイムライン設定 | 主要勢力 |
| :--- | :--- | :--- | :--- |
| ドラゴンボール ゼノバース | トキトキ都 | 様々な年代・時代 | タイムパトロール |
| ドラゴンボール ゼノバース2 | コントン都 | 魔神ドミグラ後の時代 | タイムパトロール |
| ドラゴンボール ゼノバース3 | ニュー・シティ | Age 1000（148年後） | グレートサイヤマンスクワッド（GSスクワッド） |

この構造的な変化により、カスタムサイヤ人は、より広大な複数タイムラインの紛争に飛び込む前に、身近な世界とより直接的かつ個人的な関わりを持つことになります。初期のトレーラーでは、プレイヤーキャラクターがブレットと交流したり、未来的な乗り物を運転したり、カプセルコーポレーションの指示のもとで局所的な都市の緊急事態に対応したりする様子が描かれています。

## 次世代のカスタマイズとサイヤ人のソウルボンディング

**ドラゴンボール ゼノバース3 サイヤ人**のカスタマイズは、もはや髪型や防具の色を選ぶだけではありません。ディンプスは**ソウルボンディング（Soul Bonding）**と呼ばれる画期的なシステムを導入します。このシステムは、『ドラゴンボール ファイターズ』や『ドラゴンボール ザ ブレイカーズ』といった過去のタイトルの要素を組み合わせた、直接的な進化形です。

### ソウルボンディングの仕組み
過去の作品では、プレイヤーは特定の師匠のもとで修行し、その得意技を解放する必要がありました。しかし、今作ではZ戦士たちが「魂（ソウル）」として存在します。これらの伝説的なキャラクターたちと精神的なつながり、すなわち「ソウルボンド」を結ぶことで、プレイヤーのカスタムサイヤ人は彼らの戦闘スタイル、パッシブ特性、そして究極技を受け継ぐことができます。

このシステムにより、かつてないほどのカスタマイズが可能になります。例えば、**ドラゴンボール ゼノバース3 サイヤ人**は、孫悟空のソウルと交信して「かめはめ波」や「界王拳」をマスターしたり、ベジータと交信して壊滅的な威力を持つ「ファイナルフラッシュ」を放ったりすることができます。特定の戦士との絆が深まるほど、彼らの技をより効果的に繰り出すことができるようになり、プレイヤーごとにパーソナライズされた成長ルートを提供します。

| 結束するソウル | 主なステータス上昇 | 解放される代表スキル | 最適な戦闘スタイル・シナジー |
| :--- | :--- | :--- | :--- |
| **悟空** | 気力回復＆最大体力 | 超かめはめ波 / 界王拳 | バランス / ラッシュダウン |
| **ベジータ** | 気弾系必殺技ダメージ | ファイナルフラッシュ / プロミネンスフラッシュ | 遠距離 / 気弾連打 |
| **悟飯** | 打撃系必殺技ダメージ | 魔閃光 / ビースト悟飯コンボ | 近接 / コンボ重視 |
| **未来トランクス** | 剣攻撃速度＆スタミナ | バーニングアタック / 剣スキル | 武器使用ハイブリッド |
| **ピッコロ** | 体力自動回復＆防御力 | 魔貫光殺砲 | タンク / 防御型 |

コミュニティの報告によると、ソウルボンディングはカスタムキャラクターのオーラの見た目も変化させると言われています。ベジータと深く結ばれたサイヤ人は、より鋭くギザギザした青いオーラを放ち、悟空と結ばれた戦士は、より柔らかく伝統的な白と青のエネルギーの揺らめきを表示するようになります。

## 限界を超えた変身：超サイヤ人形態と界王拳

多くのプレイヤーにとって、**ドラゴンボール ゼノバース3 サイヤ人**をプレイする究極の目的は、象徴的な変身形態を解放することです。開発陣は、変身がこれまで以上にインパクトがあり、ドラマチックで、視覚的にも圧倒的なものになると予告しています。

公式トレーラーでは、男性のカスタムサイヤ人キャラクターが路地裏で気力を溜め、劇的に超サイヤ人に変身する様子が初公開されました。ライティング効果、飛び散る瓦礫の物理演算、リアルタイムの髪型モデルの変化などは、次世代ハードウェアのパワーを遺憾なく発揮しています。

### 予想される変身ティア一覧
前作までのプレイヤーの経験や新たなAge 1000の設定に基づくと、サイヤ人の変身ツリーは非常に多様なものになると予想されます。本作は過去10年間のドラゴンボールメディアの要素を取り入れているため、クラシックな超サイヤ人から神の領域の形態まで、幅広い変身が期待できます。

| 変身形態 | 予想される必要気力 | 主なステータス上昇 | 外見・ゲームプレイの変化 |
| :--- | :--- | :--- | :--- |
| **超サイヤ人 (1-3)** | 3～5ゲージ | 物理打撃＆気弾必殺技ダメージ | 逆立つ金髪、超2での電撃、超3での長い髪 |
| **超サイヤ人ゴッド** | 4ゲージ | 通常攻撃速度＆気力回復 | 引き締まった赤髪、炎のようなオーラ、高速バニッシュステップ |
| **超サイヤ人ブルー (SSGSS)** | 5ゲージ | 全体的な大幅ダメージ上昇 | シアンの髪、きらめくオーラ、緩やかなスタミナ減少 |
| **ビースト形態** | 5ゲージ | 極めて高い打撃ダメージ＆クリティカル率 | 伸びた灰白色の髪、赤いオーラ、高い被ダメージリスク |
| **界王拳 (サブ形態)** | 持続消費 | 速度＆スタミナ回復 | 深紅のオーバーレイオーラ、ベース形態と重複可能 |

初期のゲームプレイ分析によると、形態間の移行はシームレスに行われます。開発陣は、別の姿に変身することがバトル中のボス級のイベントのように感じられるよう、細心の注意を払っています。超サイヤ人状態を発動すると、その衝撃波自体が周囲の敵を吹き飛ばし、反撃を開始するための短い隙を作り出すことができます。

## ゼノバース3における最強サイヤ人CaCのビルド方法

今後の計画に役立つよう、プレイヤーの体験談やゲームプレイ映像を統合し、**ドラゴンボール ゼノバース3 サイヤ人**の最適な初期ステータスの概要をまとめました。好みのプレイスタイルに応じて、属性値を割り振り、ソウルボンドを慎重に選択することで、キャラクターの潜在能力を最大限に引き出しましょう。

### 1. 気弾スナイパー（遠距離特化型）
このビルドは、敵との距離を保ち、素早く気力を溜め、強力なエネルギー波で相手を圧倒することに焦点を当てています。
*   **優先ステータス:** 気弾系必殺技、最大気力、通常攻撃
*   **推奨ソウルボンド:** ベジータまたはフリーザ（気力生成の高速化）
*   **主要スキル:** かめはめ波、ギャリック砲、エネルギーチャージ

### 2. 打撃ブローラー（近接特化型）
近接戦闘を好む場合、このビルドは物理コンボのダメージを最大化し、打撃系必殺技を使用して敵のガードを崩すのに適しています。
*   **優先ステータス:** 打撃系必殺技、最大スタミナ、通常攻撃
*   **推奨ソウルボンド:** 悟飯またはピッコロ（体力・回復重視）
*   **主要スキル:** メテオクラッシュ、超龍撃拳、瞬間移動

### 3. 界王拳ハイブリッド（ハイリスク・ハイリターン型）
界王拳のサブ形態を利用して絶え間なく圧力をかけ続ける、汎用性の高いビルドです。優れたスタミナ管理が求められますが、他の追随を許さないスピードを誇ります。
*   **優先ステータス:** 最大スタミナ、通常攻撃、体力
*   **推奨ソウルボンド:** 悟空
*   **主要スキル:** 界王拳、10倍かめはめ波、仕切り直し

| 属性の優先度 | 気弾スナイパー | 打撃ブローラー | 界王拳ハイブリッド |
| :--- | :--- | :--- | :--- |
| **体力** | 中 | 高 | 高 |
| **最大気力** | 高 | 低 | 中 |
| **最大スタミナ** | 中 | 高 | 極めて高い |
| **通常攻撃** | 中 | 高 | 高 |
| **打撃系必殺技** | 低 | 極めて高い | 中 |
| **気弾系必殺技** | 極めて高い | 低 | 中 |

## よくある質問（FAQ）

### 『ドラゴンボール ゼノバース3』のサイヤ人カスタマイズは、前作と何が違うのですか？
最も大きな違いは、**ソウルボンディング（Soul Bonding）**システムの導入と、次世代ハードウェアへの移行にあります。単に師匠から技を学ぶだけでなく、**ドラゴンボール ゼノバース3 サイヤ人**は象徴的なZ戦士たちの魂と精神的に結びつき、独自のパッシブ特性、戦闘スタイル、ステータス補正を受け継ぐことができます。さらに、変身時の視覚的なクオリティ、衣服の物理演算、キャラクターモデルなどが、PS5、Xbox Series X/S、Steam向けに完全に近代化されています。

### ゼノバース3ではサイヤ人に尻尾はありますか？また、動きますか？
はい、初期のコミュニティ報告やプロモーションアートワークによると、サイヤ人のカスタムキャラクターは尻尾ありで作成可能です。尻尾が静的なアクセサリーだった過去作とは異なり、次世代の本作では、キャラクターの動き、戦闘の構え、変身などに合わせてダイナミックに反応する、完全にアニメーション化された尻尾が特徴となっています。

### 『ドラゴンボール ゼノバース3』の発売日はいつですか？
本作は日本国内および世界中で**2027年**に発売が予定されています。開発はディンプスが担当し、バンダイナムコエンターテインメントからPlayStation 5、Xbox Series X/S、PC（Steam）向けにリリースされます。

### ゲーム内に学校やアカデミーのようなシステムはありますか？
はい。ストーリーモードでは、ブルマとブレットがあなたのカスタムキャラクターをニュー・シティ内にある専門学校に入学させることが明らかになっています。ここでは、世界の仕組みを学び、リリカなどの他の訓練生と一緒にトレーニングを行い、グレートサイヤマンスクワッド（GSスクワッド）の一員として特別なミッションに挑むことになります。
