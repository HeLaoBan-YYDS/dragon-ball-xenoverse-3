鳥山明氏の作品にインスパイアされたこの格闘ゲームシリーズのファンたちは、10年以上にわたり「タイムパトロール」の物語が次なる進化を遂げるのを待ちわびてきました。ドラゴンボール ゼノバース3におけるキャラクタークリエイトは、単なるメニュー画面以上の意味を持ちます。それは、プレイヤー自身がかつてないほどの詳細さでドラゴンボールの世界観に入り込むためのチャンスなのです。続編の開発に関する噂が飛び交う中、ドラゴンボール ゼノバース3のキャラクタークリエイトの奥深さは、世界中のコミュニティにとって最も期待されている機能の一つとなっています。

ゼノバースシリーズの成功は、常にプレイヤーの主体性に根ざしてきました。前2作が強固な基盤を築いた一方で、第3作目では前世代の限界を打ち破ることが期待されています。ダイナミックな髪型の変化から、長年要望されてきた人造人間やバイオ人造人間といった種族の追加まで、その成長の可能性は計り知れません。

## 基盤：なぜキャラクタークリエイトがシリーズを定義するのか

これまでのタイトルでは、「カスタムキャラクター（CaC）」がタイムパトロールの主人公を務めてきました。このキャラクターは単なるスキンではなく、プレイヤーの戦略を体現するメカニカルな拡張機能です。来るべき続編において、ドラゴンボール ゼノバース3のキャラクタークリエイトは、過去の硬直的なテンプレートから脱却し、より流動的で現代的なRPGシステムへと進化することが期待されています。

コミュニティからは、ゼノバース2は改善されたものの、依然として「ゼノバース1.5」のような感覚が拭えないという指摘がしばしばなされてきました。3作目が成功するためには、クリエイトエンジンが最新のハードウェアを活用する必要があります。それは、テクスチャの向上、よりリアルな布の物理演算、そして長年サイヤ人の変身に付きまとってきた「固定された髪型」の問題を解消することを意味します。

## 予測される種族と初期ステータス

ドラゴンボール ゼノバース3のキャラクタークリエイトで最もエキサイティングな側面の一つは、新しい種族の可能性です。「ビッグ5」（サイヤ人、地球人、ナメック星人、フリーザ一族、魔人）の復帰は確実ですが、プレイヤーの報告によれば、さらなる多様性への高い需要があることが示唆されています。

以下は、予測される種族のアーキタイプと、それが初期ビルドにどのように影響するかを示した内訳です。

| 種族 | 主な長所 | 主な短所 | 独自のパッシブアビリティ |
| :--- | :--- | :--- | :--- |
| **サイヤ人** | 攻撃力 | 低い体力 | ゼンカイブースト（体力が低いほどパワーアップ） |
| **地球人** | バランス型 | 特出した強みがない | 気力自動回復 |
| **ナメック星人** | 高い体力/防御力 | 低い攻撃力 | 体力自動回復＆アイテム効果アップ |
| **フリーザ一族** | 高いスピード | 低い防御力 | 攻撃中のスタミナ回復 |
| **魔人** | 圧倒的な防御力 | 遅いスタミナ回復 | 「浄化」または高いダメージ耐性 |
| **人造人間 (新規)** | 無限エネルギー | 気溜め不可 | 手動チャージ不要の気力/スタミナ定常回復 |

## 高度なビジュアルカスタマイズ機能

ドラゴンボール ゼノバース3のキャラクタークリエイトにおいて、最も大きなアップグレードが期待されているのがビジュアル面です。これまでのゲームでは、顔、目、髪型の数は限られていました。コミュニティの要望リストには、「プリセット」ではなく「スライダー」という言葉が頻繁に登場します。

キャラクターの筋肉の定義や手足の長さを調整し、それが戦闘中のリーチに影響を与えるようなシステムを想像してみてください。さらに、「オーラカスタマイズ」機能は2016年以来、ファンが切望してきたものです。キャラクターのベースエネルギーの色や形状を選択できるようになれば、シリーズ史上類を見ないパーソナリティの層が加わるでしょう。

| 機能 | ゼノバース2の仕様 | ゼノバース3に期待されるアップグレード |
| :--- | :--- | :--- |
| **髪型** | 固定プリセット | ダイナミック/流動的な髪型、変身時に逆立つ髪 |
| **オーラの色** | 種族/スキルに固定 | 色やスタイルを完全にカスタマイズ可能 |
| **体型** | 身長3種/体格3種 | 身長、体重、筋肉量を調整できるフルスライダー |
| **戦闘スタイル** | 種族ごとに固定 | 伝説の師匠たちの構えを選択可能 |
| **衣装** | 一部アイテムは色固定 | すべての衣装がフルカラーカスタマイズ可能 |

## 戦闘アーキタイプと戦闘スタイル

ドラゴンボール ゼノバース3のキャラクタークリエイトでは、「戦闘スタイル」の選択をプロセスの最初に行うべきです。初期のゲームでは、基本的な攻撃の連鎖は種族と性別だけで決定されていました。しかし、現代の格闘ゲームでは、よりモジュール化された選択が可能です。

プレイヤーは、象徴的なキャラクターに影響を受けた「ベーススタイル」を選択できるシステムを望んでいます。例えば、ウイスのような洗練された動きで戦うサイヤ人と、ブロリーのような荒々しい喧嘩殺法を使うサイヤ人といった違いです。これにより、同じ種族であっても、すべてのカスタムキャラクターが個性的だと感じられるようになるでしょう。

| スタイルアーキタイプ | 特化項目 | おすすめのプレイスタイル |
| :--- | :--- | :--- |
| **打撃スペシャリスト** | 物理コンボ | 近距離での乱闘 |
| **気弾スナイパー** | 遠距離飛び道具 | ゾーンコントロール |
| **テクニカルカウンター** | パリィと反撃 | 高スキルな防御重視のプレイヤー |
| **スピードスター** | 高速移動/回避 | グラスキャノン（高火力・低耐久）ビルド |
| **タンク** | スーパーアーマー/高HP | チームバトルでの攪乱 |

## 変身システム：超サイヤ人の先へ

変身システムは、どのドラゴンボールゲームにおいても心臓部です。ドラゴンボール ゼノバース3のキャラクタークリエイトにおいて、各種族に実用的な「究極」の形態を用意することが目標となります。歴史的にサイヤ人が最も注目されてきましたが（超サイヤ人1-3、ゴッド、ブルー、身勝手の極意）、競争のバランスを維持するためには、他の種族にも同等のパワーアップが必要です。

コミュニティからは、「オレンジ・ピッコロ」や「ビースト」のような形態が、特定のキャラクターだけでなく、カスタムキャラクターでもアクセス可能であるべきだという声が強まっています。もしクリエイトスイートで「多段階変身」が可能になれば、プレイヤーは単一のスキル枠に異なる形態を割り当て、戦闘中に進化していくことが可能になるでしょう。

| 種族 | 期待される新変身 | 効果 |
| :--- | :--- | :--- |
| **サイヤ人** | 超サイヤ人4 / 我儘の極意 | 圧倒的な物理強化 / ダメージベースのスケールアップ |
| **ナメック星人** | オレンジ・ナメック星人 | 巨大化および防御力バフ |
| **地球人** | 最大出力 / ビースト | スピード上昇およびクリティカル率アップ |
| **フリーザ一族** | ブラックフリーザ | 比類なきスピードと気力効率 |
| **魔人** | 変身・擬態 | 一時的に相手のステータスを模倣する能力 |

## ステータス割り振りとビルド最適化

ドラゴンボール ゼノバース3のキャラクタークリエイトでビジュアルと種族の選択が終わると、焦点は「属性ポイント」に移ります。現在のメタでは、プレイヤーは打撃系必殺技か気弾系必殺技のどちらかに特化させる「ミニマックス（最小努力で最大効果）」なキャラクター作りを行うのが一般的です。

プレイヤーの体験に基づくと、より微細なステータスシステムには「サブステータス」が含まれるべきでしょう。単なる「体力」ではなく、「回復力」や「ダメージ閾値」などが存在すれば、オンラインランクマッチですべてのサイヤ人ビルドが同じに見えることを防げるはずです。

| 属性 | ゲームプレイへの影響 | おすすめの対象 |
| :--- | :--- | :--- |
| **最大体力** | 合計HPプール | ナメック星人やタンク |
| **最大気力** | 必殺技/究極技のゲージ数 | 気弾スペシャリスト |
| **最大スタミナ** | 回避/緊急回避の回数 | 防御重視のプレイヤー |
| **基本攻撃** | 弱/強コンボのダメージ | オールラウンダー |
| **打撃必殺技** | 物理スキルのダメージ | 乱闘派 |
| **気弾必殺技** | 気功波/弾スキルのダメージ | 遠距離アタッカー |

## コミュニティの期待と統合

[バンダイナムコエンターテインメントの公式サイト](https://www.bandainamcoent.com/)では、長年続くフランチャイズにおいてコミュニティからのフィードバックがいかに重要であるかが頻繁に強調されています。ゼノバース3に対して、コミュニティは「生活の質（QoL）」の向上を強く求めています。これには、「外見ロードアウト」を「スキルロードアウト」とは別に保存できる機能などが含まれます。

さらに、キャラクターを世界観に統合する深みも必要です。ドラゴンボール ゼノバース3のキャラクタークリエイトにおいて、プレイヤーは自身の選択がNPCの反応に影響を与えることを望んでいます。例えば、フリーザ一族のキャラクターはナメック星人から疑いの目で見られたり、サイヤ人はライバルたちから頻繁に挑戦を受けたりするべきでしょう。このような「ロールプレイ」要素は、本作を単なる格闘ゲームから、真のドラゴンボール・シミュレーターへと昇華させるはずです。

## 技術的改善：Unreal Engine 5の先へ

より強力なハードウェアへの移行に伴い、ドラゴンボール ゼノバース3のキャラクタークリエイトを支えるエンジンにはUnreal Engine 5が採用されると予想されます。これにより、『ドラゴンボール超 スーパーヒーロー』や『ドラゴンボール ファイターズ』のような美学に匹敵する、高忠実度のキャラクターモデルが実現可能になるでしょう。

NaniteやLumen技術の活用は、キャラクターが気を溜めている際、その気の色が衣装に動的に反映されることを意味します。もしキャラクターがファイナルフラッシュを放てば、周囲の環境、そしてキャラクター自身のモデルにも、その強烈な黄色い光が反射されるはずです。このレベルの没入感こそが、本作を前作と差別化する鍵となります。

## カスタマイズについての総括

フランチャイズの未来を見据えるとき、プレイヤーキャラクターこそが体験の中心であることは明らかです。ドラゴンボール ゼノバース3のキャラクタークリエイト・スイートは、シリーズの伝統を尊重しながら、ライセンスを受けたアニメゲームが達成できる限界を押し広げるような、堅牢で創造的な遊び場である必要があります。プレイヤーの体験に耳を傾け、深く意味のあるカスタマイズを実装することで、開発者はすべてのタイムパトローラーが唯一無二の伝説を作っていると感じられることを保証できるはずです。

## よくある質問（FAQ）

### ドラゴンボール ゼノバース3のキャラクタークリエイトに新しい種族は追加されますか？
公式には確認されていませんが、人造人間、バイオ人造人間、さらには魔界の住人などの追加を求めるコミュニティの声は非常に大きいです。これらが追加されれば、ドラゴンボール ゼノバース3のキャラクタークリエイトシステムで利用可能なビルドの多様性が大幅に拡大します。

### ゼノバース2からキャラクターを引き継ぐことはできますか？
歴史的に、ゼノバース2では前作のヒーローをNPCとしてインポートすることができました。ドラゴンボール ゼノバース3のキャラクタークリエイトにも同様の「レガシー」システムが搭載される可能性は非常に高いですが、バランス調整のためステータスや装備はリセットされる可能性があります。

### ゼノバース3では超サイヤ人に変身すると髪型が変わりますか？
これはプレイヤー層から最も要望されている機能の一つです。ハードウェアの向上により、ドラゴンボール ゼノバース3のキャラクタークリエイトエンジンは、変身時に髪が逆立ったり形状が変化したりするダイナミックな髪型モデルをサポートできるようになるはずで、これは色が変わるだけだった前作から大きな進歩となります。

### リリース時に何人のキャラクターを作成できますか？
過去の作品に基づくと、最初は少なくとも8つのキャラクター枠が用意され、各種族と性別の組み合わせを1つずつ作成できると予想されます。多くのファンは、ドラゴンボール ゼノバース3のキャラクタークリエイトにおいて、ゲーム内のアンロックや進行を通じて、さらに多くのスロットが解放され、さまざまなビルドの実験が奨励されることを期待しています。
