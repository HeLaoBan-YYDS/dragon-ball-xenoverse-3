## キャラクタークリエイション（CAC）の刷新：髪型、筋肉、衣装

包括的な**dragon ball xenoverse 3 wishlist**を作成する際、プレイヤーが最初に指摘するのは、時代遅れとなったキャラクタークリエイション（CAC）システムです。ゼノバース2は静的なアセットに大きく依存していたため、カスタムキャラクター同士が非常に似通ってしまうという課題がありました。

### 変化する髪型とより深いカスタマイズ

フォーラムやSNS上のプレイヤーの声では、大きな不満点として「静的な超サイヤ人の髪型」が挙げられています。現在のエンジンでは、超サイヤ人に変身しても髪の形は変わらず、色が黄色に変わるだけです。コミュニティの報告では、PC向けの「Revamp Project」のようなMODが、変身に伴う髪型の変化が完全に可能であることを証明していると頻繁に称賛されています。

変身時に実際に逆立ったり形が変わったりするカスタムヘアが必要です。さらに、出来の悪いウィッグをアクセサリーとして使用するのではなく、全種族に対して本物の高品質なヘアスタイルを提供し、フリーザ族のユニークな頭の形や、女魔人の多様なオプションも用意すべきです。

| カスタマイズ機能 | ドラゴンボール ゼノバース2の現状 | ゼノバース3のウィッシュリスト目標 |
| :--- | :--- | :--- |
| **超サイヤ人の髪型** | 色の変化のみ（静的な黄色） | 完全にアニメーション化された、逆立つ変身モデル |
| **髪型の多様性** | 非常に限定的。長年変更なし | ロースターキャラの髪型移植、MODに触発された創作 |
| **ウィッグ vs 地毛** | ウィッグが貴重なアクセサリー枠を占有 | ウィッグを選択・編集可能な髪型に変換 |
| **眉毛の同期** | 色が固定。プレミアムスタイリストが必要 | アクティブな変身に基づいた動的なカラーマッチング |

### ダイナミックな体型と筋肉の定義

現在、CACの体格を調整しても、キャラクターモデルが単に伸びたり縮んだりするだけです。これにより、リアルな筋肉量ではなく、不自然なプロポーションになってしまいます。

これを解決するために、ディンプスは専用の「筋肉スライダー」を導入すべきです。これにより、プレイヤーは筋骨隆々のパワーハウス型ブロウラーや、スリムで高速な格闘家をデザインできるようになり、初期のドラゴンボールZキャラクターとシリーズ後半のキャラクターとの明確な肉体的な違いを再現できるようになります。

### カジュアルな服とアニメーションするアクセサリー

このシリーズは格闘技に深く根ざしていますが、コミュニティのフィードバックでは、よりカジュアルな服装オプションへの強い要望が示されています。トランクスのクラシックなフーディーやカスタムストリートウェアのような衣装があれば、プレイヤーはハブワールドで自分自身をより自由に表現できます。

また、あらゆる**dragon ball xenoverse 3 wishlist**の主要な柱となっているのが、完全にアニメーション化されたサイヤ人の尻尾の実装です。ゼノバース2では、装備された尻尾は完全に静止しており、硬直しています。プレイヤーは、動き、戦闘、変身に反応して動的に揺れ動く尻尾を求めています。

## ロースターの拡大：新種族、亜種、およびベースゲームへの統合

現在の種族選択（サイヤ人、地球人、魔人、ナメック星人、フリーザ一族）は強固な基盤を築きました。しかし、真の続編では、ゲームプレイを新鮮に保つためにこれらの境界をさらに押し広げる必要があります。

### 種族の分岐パスと人造人間

ロールプレイング要素を深めるために、開発者は各種族内に分岐パスを導入すべきです。例えば、地球人を選択すると、サイバネティックな「人造人間」ブランチや「三つ目族」のパスにつながるようにします。このレベルの深みこそが、本格的な**dragon ball xenoverse 3 wishlist**において刷新された成長パスが定番となっている理由です。

```
[地球人ベース] ──┬──> [純粋な人間 / 三つ目族]
                   └──> [人造人間 / セル型人造人間]

[サイヤ人ベース] ──┬──> [純血サイヤ人]
                   └──> [混血サイヤ人]
```

| ベース種族 | 提案される亜種 / パス 1 | 提案される亜種 / パス 2 | ユニークなゲームプレイ特性 |
| :--- | :--- | :--- | :--- |
| **地球人** | 純粋な人間（天津飯スタイルの第三の目） | 人造人間（永久エネルギー/吸収型） | 気力の自動回復または吸収 |
| **フリーザ一族** | コルド大王のような巨大タイプ | チルドのような原始タイプ | 身体のサイズに基づいた独自のステータススケーリング |
| **ナメック星人** | 龍族（サポート/回復） | 戦士タイプ（暗黒/第6宇宙の悪） | 特化した体力回復または攻撃バフ |
| **魔人** | 純粋/善の魔人 | 悪/超魔人 | 独特の変幻自在な戦闘スタイル |

### DLCキャラクターのベースロースターへの統合

コミュニティがロースターに最も望んでいることを尋ねれば、すべての**dragon ball xenoverse 3 wishlist**で繰り返されるテーマは、過去のDLCのベースゲームへの統合です。ゼノバース2が発売された際、ゼノバース1のDLCキャラクターをベースロースターに賢明に含めました。ゼノバース3もこの前例に従うべきです。

「ビースト悟飯」や「FUTURE SAGA」DLCのような極めて最近の追加要素はオプションのままでも良いかもしれませんが、長年のファンのお気に入りは初日から解放されているべきです。

| キャラクター | オリジナル（ゼノバース2 DLC） | ゼノバース3でベースゲームに入れるべき理由 |

## ゲームプレイメカニクス：カスタム師匠ムーブセットとシネマティックな覚醒技

ゼノバースの戦闘は常に派手でしたが、構造的な多様性に欠けることがよくありました。より深いメカニカルなカスタマイズを導入することで、競技シーンとカジュアルシーンの両方を高めることができます。

### コンボのミックス＆マッチ

プレイヤーの間で非常に要望が多い機能は、師匠の戦闘スタイルを採用できる能力です。もし自分のCACが悟飯に訓練されているなら、悟飯の弱・強コンボを装備できるようにすべきです。

さらに一歩進んで、ベジットの素早い蹴りを弱攻撃に、身勝手の極意 悟空の一撃を強攻撃に利用するなど、コンボを自由に組み合わせることができれば、比類のない戦闘の多様性が生まれます。

### サイヤ人以外の真の覚醒形態

歴史的に、サイヤ人は変身において最大の恩恵を受けてきました。このバランスを取るために、次作では他の種族にも上位の覚醒スキルを導入する必要があります。

ナメック星人には「オレンジ・ナメック星人」や「潜在能力解放」の形態が必要であり、フリーザ一族はゴールデンにいきなり飛ぶのではなく、第二、第三、最終形態へと変身できるようにすべきです。

| 種族 | 提案される覚醒スキル | 視覚的・ゲームプレイ上の変化 |
| :--- | :--- | :--- |
| **ナメック星人** | オレンジ / 潜在能力解放 | 巨大化、オレンジ色の肌、大幅な防御バフ |
| **フリーザ一族** | 多段変身プログレッション | 戦闘中に第二、第三、最終形態をサイクル |
| **魔人** | 吸収 / 超形態 | 相手の特性を吸収、または洗練された超ブウモデルへ移行 |
| **地球人** | 武器マスタリー / マッスル形態 | 亀仙人スタイルの増量、または高度な物理武器の使用 |

## 再構築されたハブワールドとキャンペーン

コントン都はトキトキ都から大きく進化しましたが、最終的には空虚で静的な場所になってしまいました。次のハブワールドは、生き生きとしてインタラクティブで、シームレスに感じられる必要があります。

### コントン都とトキトキ都の統合

コミュニティで人気のアイデアは、トキトキ都の歴史的なランドマークとコントン都の広大な景観を統合することです。この新しいハブには、ダイナミックな天候、昼夜のサイクル、インタラクティブなミニゲームを備えるべきです。

さらに、プレイヤーは刻蔵庫（タイムネスト）の内部を含め、どこへでも即座に飛行できるべきであり、遅い歩行速度や恣意的な見えない壁に悩まされるべきではありません。

### シネマティックでリブートされたストーリー

現代のドラゴンボールメディアで魔界の描写が公式に拡大している今、ゼノバースのストーリーは完全なリブートが必要です。単にサイヤ人編をマイナーチェンジして3回目として繰り返すのではなく、未探索の「もしも（What-If）」シナリオを深く掘り下げるべきです。

ゴクウブラックならぬ「ゴハンブラック」が存在したり、プレイヤーが合体したビルスとウイスを止めなければならなかったりするキャンペーンを想像してみてください。プレイヤーにストーリーを分岐させる選択肢を与えることで、タイムパトロールは真の選択主導のアドベンチャーになります。

## よくある質問

### ドラゴンボール ゼノバース3の発売日はいつですか？
バンダイナムコの公式ランディングページによると、ドラゴンボール ゼノバース3は**2027年**にPlayStation 5、Xbox Series X|S、およびSteamで発売予定です。

### ゼノバース2のカスタムキャラクターをゼノバース3に引き継げますか？
バンダイナムコはまだセーブデータの引き継ぎを公式に認めていませんが、コミュニティは、ゼノバース1のヒーローがゼノバース2に統合されたのと同様に、以前のタイムパトローラーを伝説の師匠や歴史的人物としてインポートできる機能を強く期待しています。

### なぜ今、ドラゴンボール ゼノバース3 ウィッシュリストがコミュニティにとって重要なのですか？
2027年の発売が公式に目標とされたことで、プレイヤーは開発元のディンプスが長年のエンジン制限に対処することを望んでいます。**dragon ball xenoverse 3 wishlist**を共有することは、開発サイクル中にキャラクターカスタマイズ、ロースターの選択、戦闘メカニクスに関するプレイヤーのフィードバックを集約するのに役立ちます。

### ゼノバース3に新しいプレイアブル種族は登場しますか？
新しい種族は公式に発表されていませんが、コミュニティはRPG要素を拡大するために、プレイアブルな人造人間、セル型人造人間、および魔界の住人の追加を求めて激しいキャンペーンを行っています。
