## オンラインプレイの進化：AGE 1000で何が変わるのか？

『ゼノバース2』のコントン都から、AGE 1000の高度に発展した西の都への移行は、大きな技術的飛躍を意味します。これまでの作品では、マルチプレイヤーのハブは主に限定的な相互作用しか持たないマッチングロビーとして機能していました。開発者による初期の発表によれば、新しいハブはプレイヤーがシームレスに他者と出会える、活気ある生きた世界として機能するとのことです。

**『ドラゴンボール ゼノバース3』のマルチプレイヤー**環境のコアとなるサイクルを見ると、オープンワールド要素の統合が最も注目すべきアップグレードです。プレイヤーはその場でチームを組み、クエストを共有し、ダイナミックなワールドイベントに参加することができます。以下の表は、シリーズを通じてハブとマルチプレイヤーシステムがどのように進化したかをまとめたものです。

| 機能 | ドラゴンボール ゼノバース2 | ドラゴンボール ゼノバース3 |
| :--- | :--- | :--- |
| **主要ハブワールド** | コントン都（インスタンス形式） | AGE 1000の西の都（オープンワールド） |
| **ハブの最大人数** | 限定的なロビーインスタンス | 拡張されたシームレスなMMORPG型サーバー |
| **ワールド内の相互作用** | 静的なNPCと基本的なエモート | ダイナミックイベント、ゲーム内チャット、協力探索 |
| **ミッション開始方法** | 専用のクエストカウンター | オープンワールドから戦闘ゾーンへのシームレスな移行 |
| **ソーシャル機能** | 基本的な定型文とロビーアバター | 高度なチャット、クランシステム、インタラクティブなエモート |

このオープンワールド構造への移行により、自然発生的な協力プレイが可能になります。マッチングメニューで待機する代わりに、西の都で仲間に近づき、即座にグループを結成できるようになります。

## 協力ミッションとグレートサイヤン隊

協力プレイは常にゼノバース・シリーズの核心であり、第3作目ではこれをさらに一歩進めています。プレイヤーはAGE 1000の平和を維持するための組織「グレートサイヤン隊」の一員として導かれます。仲間と共に戦いながら、ドラゴンボールの世界の未来を左右する物語主導のイベントに立ち向かうことになります。

ディンプスの開発チームは**『ドラゴンボール ゼノバース3』のマルチプレイヤー**の接続性に大きな重点を置いているため、協力ミッションはより戦術的な設計になっています。高難易度のボスを攻略するには、キャラクターのクラス、究極技、そしてサポートアビリティを調整し、連携する必要があります。

| 協力ミッションの種類 | 予想プレイヤー数 | 主な目的 | 主な報酬 |
| :--- | :--- | :--- | :--- |
| **タイムパトロール作戦** | 1〜3人 | 歴史の改変を修正し、ストーリーを進める | 経験値、ストーリー進行、基本装備 |
| **西の都パトロール** | 2〜4人 | 突発的な敵の侵攻からオープンワールドのハブを守る | 希少なクラフト素材、コスメティックアイテム、通貨 |
| **グレートサイヤン隊レイド** | 6人以上 | 連携ギミックを駆使して巨大ボスを撃破する | レジェンダリー装備、限定称号、専用スキル |
| **パラレルクエスト** | 1〜3人 | 特定の条件下でifシナリオのバトルをクリアする | スキル解放、キャラクター衣装、Zソウル |

コミュニティの報告によると、これらのレイドにはマルチステージのボス戦が含まれ、チームを分割して異なる目的を同時にこなす必要があるようです。例えば、一つのグループがバリアを守っている間に、別のグループが巨大ボスに直接ダメージを与えるといった展開が予想されます。

## マルチプレイヤーにおけるキャラクターカスタマイズとチームシナジー

ゼノバース・シリーズの大きな魅力は、自分だけのカスタム「タイムパトローラー」を作成できることです。今作では、キャラクターカスタマイズがマルチプレイヤーでの役割（ロール）に直接影響を与えるよう拡張されています。全員が最大ダメージを出すことだけに集中するのではなく、**『ドラゴンボール ゼノバース3』のマルチプレイヤー**レイド向けにビルドを最適化することが不可欠になります。

種族ごとに異なる利点があり、チーム構成を補完し合います。ハイレベルなPvEや対戦PvPのチームを組む際、これらの種族特性のバランスを取ることが勝利の鍵となります。

| 種族 | 特徴的なステータス | マルチプレイでの理想的な役割 | 固有のアビリティ |
| :--- | :--- | :--- | :--- |
| **サイヤ人** | 高い攻撃力、低い体力 | メインアタッカー (DPS) | 全開パワー（体力が減るとステータス上昇） |
| **ナメック星人** | 高い体力、自動回復 | タンク / サポート | アイテム効率とパッシブな体力回復 |
| **魔人** | 高い防御力、気力回復 | 防衛 / 妨害 | 気力最大時に防御力が大幅上昇 |
| **フリーザ一族** | 高いスピード、気弾威力 | 遠距離DPS / スピードタイプ | 体力が減るほどスピードが増加 |
| **地球人** | バランス型、技ゲージ自動回復 | ユーティリティ / 万能型 | 技ゲージ（気）が時間経過で自動チャージ |

例えば、ナメック星人のプレイヤーが前線で敵の注意を引くタンク役を務め、その間にサイヤ人が遠距離から強力なファイナルフラッシュを溜めるといった連携が可能です。このようなシナジーにより、チーム構成は前作以上に重要になります。

## 競技的なPvPとランクマッチ

対人戦を好むプレイヤーのために、競技モードも大幅な刷新が行われています。コミュニティの報告によれば、**『ドラゴンボール ゼノバース3』のマルチプレイヤー**用ネットコードはゼロから構築されており、前作までの課題であった遅延（レイテンシ）を最小限に抑えるよう設計されています。

西の都内には専用のアリーナが設置され、プレイヤーはそこからランクマッチやカジュアルマッチのキューを入れることができます。観戦モードやゲーム内チャットにより、コミュニティが独自にトーナメントを開催することも容易になります。

| PvPゲームモード | チーム人数 | 試合ルール | ランキングシステム |
| :--- | :--- | :--- | :--- |
| **ランクデュエル** | 1v1 | レベル補正ありの標準的な戦闘 | グローバルリーダーボードと階級制 |
| **チームバトル** | 2v2 または 3v3 | 勝ち抜きまたは交代制のコンバット | カジュアルおよびランクキュー |
| **エンドレスバトル** | 最大6人 | 勝者残り形式のロビー、観戦チャットあり | ランクなし / 練習用 |
| **ヒーローコロシアム** | 1v1 | 収集したフィギュアを用いた戦略バトル | 独自の戦術ランキング |

ゲーム内購入やシーズンパスの導入も予想されており、ランクPvPの成果に応じて限定の見た目報酬（コスメティック）が手に入るなど、プレイヤーが上位を目指す動機付けも強化される見込みです。
