本作の正式発表はゲーム業界に大きな衝撃を与えました。公式ストアページや初期のプレスリリースによると、ゲームは2027年に発売予定です。従来の『ドラゴンボールZ』や『ドラゴンボール超』のタイムラインにおける歴史の改変を修正することに焦点を当てていた過去作とは異なり、本作はまったく新しい道を切り開いています。 

物語の舞台は、ドラゴンボール宇宙において未開拓の未来の時代である「AGE 1000」です。この未来のタイムラインには、高度に発達した環境が描かれており、特に活気にあふれ完全に再現された西の都が目を引きます。プレイヤーは、自分で自由にカスタマイズしたヒーローとしてこの世界に降り立ち、シリーズの生みの親である伝説の漫画家・鳥山明氏が手がけたオリジナルキャラクターたちと交流することになります。

初期のティザー映像によると、プレイヤーは「グレートサイヤン隊（Great Saiyan Squad）」の一員として迎え入れられます。この名誉あるスタート地点から、強力な仲間たちと共に戦い、未来社会の根幹を脅かす謎を解き明かしていきます。今回の作品では、高速バトルとオープンワールド探索を融合させた、没入感の高いアクションアドベンチャー体験に重点が置かれています。

## AGE 1000の探索：新たな舞台とストーリー要素

「AGE 1000」への移行は、本シリーズにおける最もエキサイティングなストーリー上の変化と言えるでしょう。ラディッツ、フリーザ、セルとの戦いを三度繰り返すのではなく、プレイヤーは従来のタイムラインを超えて進化した世界を冒険することになります。西の都は、そびえ立つ摩天楼、高度なカプセル技術、そして調和して暮らす多様な宇宙人たちで満ちた、広大な未来のメトロポリスへと成長しています。

ファンの推測や類似のRPGにおけるプレイヤーの経験から、西の都はシームレスで広大なハブワールドとして機能すると考えられています。エリアごとに区切られていた前作のコントン都とは異なり、この新しいハブでは、ダイナミックな天候の変化、昼夜のサイクル、そしてサイドクエストや修業、ユニークなアイテムを提供してくれるインタラクティブなNPCが配置されると予想されています。

### AGE 1000における主要なストーリー要素：
*   **グレートサイヤン隊:** キャンペーンの序盤でプレイヤーが加入する平和維持部隊。
*   **鳥山明氏の遺産:** 故・鳥山明先生本人が描き下ろしたキャラクターデザインや世界観の設定により、ゲームには本物の「ドラゴンボールの魂」がしっかりと受け継がれています。
*   **まったく新しい脅威:** タイムパトロールが後退し、AGE 1000固有の新しい勢力や宇宙規模の脅威が主な敵として立ちはだかります。

多くのファンは、今後登場する**ドラゴンボール ゼノバース3 体験版**において、これらのオープンワールド要素の一部が公開され、西の都の限定エリアを探索したり、導入部分のサイドクエストをいくつかクリアしたりして、新しい環境の圧倒的なスケール感を体験できるようになることを期待しています。

## シリーズの進化：過去作との比較

このゲームに対する期待がこれほどまでに高い理由を理解するには、シリーズがどれほど進化しているかを見る必要があります。初代『ドラゴンボール ゼノバース』は2015年に発売され、続く第2作は2016年にリリースされました。『ゼノバース2』は発売後も長年にわたりDLCやアップデートを受け取ってきましたが、そのコアエンジンと設計思想は10年以上前のものです。

第3作目となる本作は、現代のハードウェアの性能を最大限に活かすためにゼロから構築されています。開発チームは高度なレンダリング技術を駆使し、鳥山明氏の特徴的なアートスタイルを、息をのむような3Dの忠実度、滑らかに破壊される環境、そしてシームレスなトランジションアニメーションとともに現代の世代へと蘇らせています。

| 機能比較 | ドラゴンボール ゼノバース2 | ドラゴンボール ゼノバース3 |
| :--- | :--- | :--- |
| **エンジン＆グラフィック** | 前世代のカスタムエンジン、静的ライティング | 現代の高精度エンジン、動的物理演算 |
| **ハブワールドの設計** | ロード挟むエリア分割型（コントン都） | シームレスなオープンワールド探索（西の都） |
| **タイムラインの焦点** | DBZ/DBSの歴史改変の修正 | まったく新しい未来の探索（AGE 1000） |
| **バトルの深み** | 標準的な弱/強スタミナブレイクシステム | 強化された戦術的戦闘、環境との相互作用 |
| **協力プレイの統合** | ロビーベースのミッションとレイド | シームレスな途中参加・離脱可能なワールドイベント |

コミュニティの報告によると、戦闘システムはよりレスポンスが良くなるよう再構築されています。プレイヤーは、近接格闘とド派手な気弾攻撃のより流れるようなコンボを期待でき、環境破壊表現にも新たな焦点が当てられています。西の都のビルに敵を叩き込めば、そのビルはリアルな構造ダメージを受け、戦闘中その破壊された状態が維持されます。

## よくある質問

### 『ドラゴンボール ゼノバース3』の体験版はいつリリースされますか？

### 『ドラゴンボール ゼノバース3』体験版のデータは製品版に引き継げますか？
近年のバンダイナムコのRPGの多くでは、無料の公開体験版で作成したセーブデータを発売時に製品版へ引き継ぐことができます。ただし、初期の「クローズドネットワークテスト」や「ベータテスト」のデータは、発売日にすべてのプレイヤーが公平にスタートできるよう、通常はリセットされます。詳細なルールは、体験版のリリースが近づいた際に開発元から発表される予定です。

### 『ドラゴンボール ゼノバース3』の体験版は無料でプレイできますか？
はい、従来のリリースモデル通りであれば、**ドラゴンボール ゼノバース3 体験版**はSteam、PlayStation、Xboxのすべてのプレイヤーに向けて完全に無料で配信されます。無料体験版をプレイするためにゲームを予約する必要はありませんが、予約購入者向けに一部のベータテストへの先行アクセス権が提供される可能性はあります。

### 新作ではどの種族を使ってプレイできますか？
