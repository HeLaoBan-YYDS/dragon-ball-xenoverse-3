## 新たなゲームプレイメカニズムと戦闘システム

戦闘は常に『ゼノバース』シリーズの核心であり、開発陣はゲーム体験を新鮮に保つために重要な調整を行っています。プロモーション映像からは、純粋なアリーナ対戦から「アドベンチャーと対戦のハイブリッド」スタイルへの移行が見て取れます。プレイヤーは、環境破壊要素やインタラクティブなステージを重視した、より流動的な戦闘システムを期待できるでしょう。

トレーラーを分析した多くのファンは、アニメーションの移行が滑らかになったことや、新たな「ディフレクション（弾き）」メカニズムの実装を指摘しています。コンボを回避するために単にバニシングステップを踏むだけでなく、リアルタイムで反撃に転じることができる防御オプションがプレイヤーに与えられます。

| メカニズム名 | 操作・仕組み | ゲームプレイへの影響 |
| :--- | :--- | :--- |
| **ダイナミック・ディフレクション** | タイミングよくボタンを押すことで、飛んでくる気弾を相手に跳ね返す。 | 防御の選択肢が増え、技量（技ゲージやバニシング）への依存度が下がる。 |
| **エンバイロメンタル・インパクト** | 重攻撃によって相手を建物に叩きつけ、ステージの形状を変化させる。 | アニメを忠実に再現した、より没入感のある戦闘体験を提供。 |
| **タッグチーム・シナジー** | AIや協力プレイのパートナーと連携して強力な超必殺技を繰り出す。 | チームワークと戦略的なキャラクター選択が有利に働く。 |
| **アウェイクン・オーバードライブ** | ステータスは大幅に向上するが、スタミナが徐々に減少するハイリスク・ハイリターンな状態。 | 長期戦における戦術的な深みが増す。 |

これらの変更は、カジュアルなボタン連打と、競技性の高い対戦格闘ゲームのメカニズムとのギャップを埋めることを目的としています。先行体験版をプレイした人の感想によると、初心者にも親しみやすい操作感でありながら、熟練プレイヤーにも十分なやり込み要素が用意されているとのことです。

## キャラクターロースターとカスタマイズの進化

本シリーズの最大の魅力の一つは、アバタ作成（Create-a-Character = CAC）システムです。トレーラーでは、より詳細になった3Dモデル、衣装のリアルな物理挙動、そして種族ごとのユニークな特性など、カスタマイズ機能の大幅なアップグレードが示唆されています。

プレイヤーの体験談やトレーラーを巡る議論によると、カスタマイズ要素は大規模な見直しが行われており、新しいプレイアブル種族の追加や、より深いサブクラスシステムの導入が期待されています。

| 種族 / 勢力 | 固有の特性 | カスタマイズのアップグレード |
| :--- | :--- | :--- |
| **サイヤ人** | 全開パワー（体力が低い時に攻撃力上昇） | 尻尾の物理挙動、超サイヤ人時のオーラ色のカスタマイズ。 |
| **地球人** | 気の自動回復 | 多彩な格闘スタイル、高度なテクノロジー武器。 |
| **ナメック星人** | 高い体力回復力、巨大化 | 触覚の形状カスタム、多様な皮膚テクスチャパターン。 |
| **魔人** | 高い防御力、回復ボーナス | 体型のカスタマイズ、ユニークな吸収スキル。 |
| **フリーザ一族** | 高い移動速度、麻痺効果付きの気弾 | 生体装甲のカスタマイズ、メタリックな光沢スライダー。 |

伝説的な師匠たちから格闘スタイルを組み合わせて継承できる可能性もあり、パーソナライズの幅がさらに広がります。プレイヤーはキャラクターの見た目だけでなく、戦闘中の動きや立ち振る舞いまで定義できるようになります。
