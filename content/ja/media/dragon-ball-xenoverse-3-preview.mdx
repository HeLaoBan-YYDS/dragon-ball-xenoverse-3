## 次世代ビジュアルとエンジンの進化

続編において最も期待されている要素の一つが、グラフィックの飛躍的な進化です。現行世代機とPCのみをターゲットにすることで、ディンプスはこれまで不可能だった高度なレンダリング技術を活用できます。ダイナミックに破壊される環境、リアルな気（オーラ）の物理挙動、そしてシネマティックなカットシーンからリアルタイムのゲームプレイへのシームレスな移行などが期待されます。

本プレビューでは、これらのビジュアル面のアップグレードが戦闘全体の流れにどのような影響を与えるかを検証します。現代のコンソールの余剰な処理能力により、より高いフレームレートと高速なロード時間が実現し、これはテンポの速いマルチプレイヤーバトルにおいて極めて重要になります。

## マイキャラ（CaC）と種族のカスタマイズ

『ゼノバース』シリーズの核心は、常にマイキャラ（CaC：Custom Character）システムにあります。プレイヤーは自分だけのタイムパトローラーを作成し、伝説のZ戦士たちのもとで修業することを楽しんでいます。この『ドラゴンボール ゼノバース3』プレビューで言及しているように、現代のハードウェアへの移行により、カスタマイズオプションが大幅に拡張されることが期待されます。

長年のプレイヤーの経験から、ファンはより多様な体型、ユニークな戦闘スタイル、および種族固有の変身（覚醒技）を求めていることが分かっています。例えば、サイヤ人は発売時点でついに「超サイヤ人4」や「超サイヤ人ゴッド」を使用できるようになり、ナメック星人やフリーザ一族などの他の種族も、独自のユーティリティ性の高い変身形態を獲得する可能性があります。

| プレイアブル種族 | 固有の特徴 | 期待される変身（覚醒技） |
| :--- | :--- | :--- |
| **サイヤ人** | 高い攻撃性能、低い体力 | 超サイヤ人1-4、超サイヤ人ゴッド、ビースト |
| **地球人** | バランスの取れたステータス、気の自動回復 | 如意棒マスター、身勝手の極意”兆” |
| **ナメック星人** | 高い体力、体力の自動回復 | 巨大化、オレンジ・ナメック星人 |
| **フリーザ一族** | 高いスピード、気弾特化 | ゴールデン化、ブラックフリーザ |
| **魔人** | 高い防御力、スタミナ依存の回復 | 浄化、スリム形状 |

これらの種族に加えて、コミュニティの報告では、人造人間やバイオ人造人間といった新しいプレイアブル種族を望む強い声が上がっています。公式には確認されていませんが、これらのオプションが追加されれば、リプレイ性とカスタマイズの奥深さは大幅に向上するでしょう。

## バトルメカニクスと快適性（QoL）の向上

戦闘において、本シリーズは手軽に遊べるアリーナアクションと、奥深いコンボカスタマイズのバランスを常に保ってきました。しかし、ベテランプレイヤーからは、バランス調整、ネットコード、および対戦プレイの改善が長年求められてきました。この『ドラゴンボール ゼノバース3』プレビューでは、ディンプスがバランス調整において注力する可能性が高い分野に焦点を当てます。

具体的には、遅延のないオンラインマッチを保証するために、ロールバックネットコードの実装が期待されています。さらに、スタミナブレイクの仕組みを調整し、防御オプションをより直感的にすることで、カジュアルプレイヤーと競技プレイヤーの間のギャップを埋めることができるでしょう。

| バトル要素 | 現行システム | 提案される次世代システム |
| :--- | :--- | :--- |
| **オンラインネットコード** | ディレイ方式（ラグが発生しやすい） | ロールバック方式（スムーズな接続） |
| **スタミナ管理** | 厳しいスタミナブレイク | ダイナミックな回復とパリィ判定の猶予 |
| **コンボカスタマイズ** | 種族ごとの固定ルート | モジュール式の弱/強攻撃ルート入力 |
| **究極技** | 発動時の高い無敵時間 | カウンター手段とクラッシュメカニクス |

これらの変更を導入することで、マルチプレイヤー環境はより競争力が高まり、やりがいのあるものになるでしょう。また、プレイヤーが一部の強すぎるテンプレ構成に依存するのではなく、さまざまなビルドや戦略を試すきっかけにもなります。

## コミュニティの期待と2027年への道のり

公式発表以来、コミュニティは憶測で持ちきりです。前作のプレイヤー体験に基づくと、コミュニティの最大の要望は、よりインタラクティブなハブワールドです。固定された「コントン都」ではなく、プレイヤー同士が交流し、取引し、トーナメントを企画できる、活気あるソーシャルスペースが求められています。

さらに、ファンはアイテム集め（周回）のプロセスがより効率化されることを望んでいます。前作におけるレアアイテム、スキル、超ソウルの獲得クエストは、ストレスのたまるランダム要素（RNG）に依存しているとして批判されることがよくありました。天井システムや、レアドロップを直接購入できるショップがあれば、全体的なプレイヤー体験は大幅に向上するでしょう。

2027年の発売時期に近づくにつれ、バンダイナムコからキャラクタートレーラー、ゲームプレイ映像、オープンベータテストなどが公開されることが予想されます。これらのベータテストに参加することは、ディンプスに直接フィードバックを送り、最終的な製品の形を整える上で、プレイヤーにとって非常に重要になります。

## よくある質問

### ゲームの公式発売日はいつですか？
パブリッシャーの公式サイトによると、ゲームは2027年内の発売を予定しています。正確な月日はまだ発表されていませんが、ファンはすでに希望のプラットフォームでゲームをウィッシュリストに追加することができます。

### 対応プラットフォームは何ですか？

キャラクターの引き継ぎに関する公式な発表はまだありませんが、コミュニティの噂によると、同じプラットフォームで前作のセーブデータを持っているプレイヤーには、特別なレガシー報酬やカスタマイズアイテムが提供される可能性があると示唆されています。
