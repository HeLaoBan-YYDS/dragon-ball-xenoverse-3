## 视觉与主题建议

我建议你把 **默认主题设为深色**。原因很简单：Anime Vanguards 的品牌感天然更适合“深底 + 霓虹高对比 + 发光描边”的方向，这样最容易承接 Roblox 动漫塔防用户熟悉的视觉语境，也更利于把 Logo、按钮和首屏 CTA 拉出来。亮色主题可以保留，但更适合作为备用或后台文档模式。

推荐主色方向：**霓虹紫**作为品牌主色，**电光青**作为高亮辅助色。前者更像主 Logo / 页签 / 激活态；后者更适合 hover、高亮边框、状态、标签和次 CTA。

按你要求的格式，我给你一套可直接贴进变量系统的值：

```css
/* 导航页主题色 - 亮色主题 */
--nav-theme: 278 78% 56%;  /* neon-violet */
--nav-theme-light: 191 95% 64%;  /* electric-cyan */

/* 导航页主题色 - 暗色主题 */
--nav-theme: 278 78% 56%;  /* neon-violet */
--nav-theme-light: 191 95% 64%;  /* electric-cyan */
```

如果你还要往下补一层语义，我建议这样理解：

- `--nav-theme`：品牌主按钮、激活 Tab、Logo 发光主色  
- `--nav-theme-light`：悬停、Tag、选中描边、次级按钮高亮  

在页面落地上，首页背景建议走 **near-black / indigo-black**，卡片面板用 **低饱和蓝灰**，再让紫青两色去拉开层次。这样既能保住“动漫塔防”的冲击感，也不会让整站像花哨活动页。
